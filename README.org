** Installation

Add this line to your application's Gemfile:

#+BEGIN_SRC ruby
  gem 'capistrano-docker',
    git: "https://github.com/goldstarlearning/capistrano-docker.git"
#+END_SRC

And then execute:

#+BEGIN_SRC sh
bundle
#+END_SRC

** Usage

Add this line to your capistrano configuration:

#+BEGIN_SRC ruby
  require "capistrano-docker"
#+END_SRC
